
# [npm]安装

*快捷方式：用`i`替代`install`*

## 本地安装

安装并写入`package.json`的`dependencies`

    npm install xxx –save

安装并写入`package.json`的`devDependencies`

    npm install xxx –save-dev

安装但不写入`package.json`

    npm install xxx

安装`package.json`文件中的依赖到本地`node_modules`文件夹

    npm install

## 全局安装

    npm install -g xxx
