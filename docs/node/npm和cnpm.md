
# npm和cnpm

## npm vs cnpm

`npm`(`node package management`)是`node.js`的包管理器

`cnpm`是淘宝提供的`npm`镜像

>这是一个完整 npmjs.org 镜像，你可以用此代替官方版本(只读)，同步频率目前为 10分钟 一次以保证尽量与官方服务同步。

## 安装cnpm

安装`cnpm`

    npm install -g cnpm --registry=https://registry.npm.taobao.org

安装完成后使用`cnpm`代替`npm`命令
