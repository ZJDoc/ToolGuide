
# Commit Message Editor

`Commit Message Editor`提供了格式化提交信息模板

## 安装

在插件库搜索`Commit Message Editor`进行安装

## 操作

提供了两种模式，一个是直接编辑整个提交信息；另一个是使用模板进行编辑
